# stt-template #

This is a Robot Framework project template that is designed to be used with the Story to Test (STT) technique for writing tests



### Installation and Use ###

* Start with a Given/ When/ Then user story
* Clone the STT Template repo
* Write a test case with Given/ When/ Then clauses - update template.robot
* Write ‘import’ level test fragments for each clause - update All.Imports.resource
* Write ‘object’ level test fragments for each import fragment - create object resource files
* Modify the test settings for your application - update dev.resource with application URL and Title
* Verify that the test runs
